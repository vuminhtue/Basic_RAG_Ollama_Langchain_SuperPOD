This is 2nd workshop on the series of LLM on SuperPOD
Pre-requisite for this workshops are the 2 workshops:
- AI on SuperPOD
- LLMs on SuperPOD

This workshop introduce basic RAG technique using Ollama, Langchain running locally on SMU SuperPOD.
- Basic Langchain for Prompt Engineer and LCEL
- Memory on Langchain
- Various document loading types
- Various document splitting technique to chunks
- Tokens, Tokenization and Contextual embedding
- Semantic search, Reranker, Retrieval
- Basic RAG workflow

Create conda env and install requirement:

```
$ pip install -r requirements.txt
```
